# Task 5 – Data Processing with PySpark and PostgreSQL

## Описание проекта
Проект `task_5` демонстрирует обработку данных с использованием **PySpark** и взаимодействие с **PostgreSQL**. Он включает несколько обработчиков для различных сценариев: основной обработчик, обработчик решений и специфический PySpark-обработчик.

---

## Структура проекта

```
task_5/
├── handlers/ # Каталог с основными обработчиками
│ ├── init.py # Инициализация пакета
│ ├── main_handler.py # Основной обработчик данных
│ ├── solution_handler.py # Обработчик для решения задач
│ └── spark_handler.py # Обработчик на PySpark
├── jars/
│ └── postgresql-42.7.3.jar # JDBC драйвер для PostgreSQL
├── log4j2.properties # Конфигурация логирования для Spark
├── main.py # Точка входа проекта
└── README.md # Документация проекта

```
